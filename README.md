已安装驱动的用户，在后续版本中无需重复安装。

3. 使用工具箱
运行工具箱并按照屏幕上的提示进行操作。
刷写过程中请耐心等待，切勿中断操作。

4. 自定义内核刷写
若您拥有定制内核文件，请将其重命名为 "boot.img" 并放入工具箱文件夹中。
运行工具箱，当提示检测到 "boot.img" 文件时，按照指示选择1进行刷写。
